README

Thisp project is a Ruby online catalog. Regular users can make purchase orders. Admin users can manipulate resources such as Products, Users and Orders.

This was extracted from a tutorial and is fully functional.
Must have ruby on rails. 

Clone the repo and run:

bundle install
rake db:migrate
rake db:seed
rails s



Access the app with a browser in localhost:3000.
You can use sample user javier pass javier.
